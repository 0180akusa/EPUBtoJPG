借助[Calibre](https://calibre-ebook.com/ja/download) 和 [插件](https://github.com/noDRM/DeDRM_tools/releases)，我们可以轻易地将`.azw`格式的Kindle图书转换为`.epub`格式。

将`.epub`文件的后缀名改为`.zip`后，我得到了一张`cover.jpg`，一个包含`.jpeg`图片的文件夹等等。

本程序将这一过程自动化运行。
